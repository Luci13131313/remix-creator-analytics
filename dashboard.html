Creator Analytics

Paste your Remix API key to load your games' stats.

🔒 Your key never leaves this machine. It's sent only to your own local server on localhost — never to us, never to any third party, never to the browser or localStorage. This tool is open-source; you can read every line before pasting. The only place it's ever written is the local .env above (and only if you leave that ticked).
Prefer not to type it here at all? Put REMIX_API_KEY=sk_live_… in a .env file next to the server and just refresh — that's the safest. Get a key at remix.gg/api.
